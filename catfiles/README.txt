Place your Veado PNGtuber character assets here.

Supported formats: PNG, JPG, GIF, WebP, SVG, APNG

The system will auto-scan this directory and suggest state mappings
based on filenames. For best auto-detection, name files like:
  - idle.png
  - talk.png / mouthOpen.png / mouthClosed.png
  - happy.png
  - sad.png
  - angry.png
  - warning.png
  - glitch.png
  - neutral.png
  - smug.png
  - disappointed.png

You can also use any naming convention and map manually via
the Admin > Character Asset Mapper UI.
